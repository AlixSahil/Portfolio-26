Certification issuer logos
==========================

Drop each issuer's logo here and the certification cards will use it automatically.

Expected file names (either .svg or .png works — .svg preferred for crispness):

  oracle.svg     (or oracle.png)     -> Oracle certs
  google.svg     (or google.png)     -> Google IT Automation
  deloitte.svg   (or deloitte.png)   -> Deloitte Data Analytics
  mckinsey.svg   (or mckinsey.png)   -> McKinsey.org Forward Program

Notes:
- Square-ish, transparent-background logos look best (they sit on a white tile).
- The card tries /logos/<name>.svg, then .png, then a live logo service, then a
  colored monogram — so it always shows something even before you add a file.
- No code change needed once the files are here; just match the names above.
